Instructions détaillées
1

🗓️ Google Calendar

Paramètres → Autres agendas → + → Via une URL → coller le lien webcal:// → Ajouter un agenda.
Ou cliquer directement le bouton S'abonner ci-dessus.

2

🍎 Apple Calendar (Mac)

Menu Fichier → Nouvel abonnement à un agenda → coller l'URL → fréquence : Chaque heure.
Sur iPhone : Réglages → Calendrier → Comptes → Autre → Ajouter calendrier avec abonnement.

3

📧 Outlook Web (office.com)

Calendrier → Ajouter un agenda → S'abonner depuis le web → coller l'URL → Importer.

4

💻 Outlook Desktop

Fichier → Paramètres du compte → Calendriers Internet → Nouveau → coller l'URL → OK.

5

📱 Autres apps (Thunderbird, Proton…)

Si webcal:// n'est pas reconnu, remplacer par https:// dans le lien.
