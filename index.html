⚡ Le lien webcal:// ouvre automatiquement votre application de calendrier

🔄 Comment ça fonctionne : Tous les horaires sont stockés en UTC absolu (format RFC 5545 avec suffixe Z). Votre application de calendrier convertit automatiquement chaque match à votre heure locale, y compris les changements d'heure d'été (DST). Exemple : La finale (19 juillet, 19:00 UTC) s'affiche à 21h à Paris, 20h à Abidjan, 15h à New York, 04h à Tokyo (J+1).

Instructions détaillées

1

📅 Google Calendar

Ouvrir Google Calendar → Autres agendas → + → Via une URL
Coller l'URL webcal:// et cliquer Ajouter un agenda.
Alternative : cliquer directement sur le bouton « S'abonner » ci-dessus.

2

🍎 Apple Calendar (Mac)

Menu Fichier → Nouvel abonnement à un agenda…
Coller l'URL et choisir la fréquence de rafraîchissement → Chaque heure.
Sur iPhone/iPad : Réglages → Calendrier → Comptes → Autre → Ajouter cal. avec abonnement.

3

📧 Outlook Web (office.com)

Calendrier → Ajouter un agenda → S'abonner depuis le web
Coller l'URL webcal:// (ou https://) → Importer.

4

💻 Outlook Desktop (Windows/Mac)

Menu Fichier → Paramètres du compte → Calendriers Internet → Nouveau…
Coller l'URL, valider. Les matchs apparaissent dans un agenda séparé.

5

📱 Autres applications (Thunderbird, Proton, etc.)

Utiliser le lien https:// (pas webcal://) si l'app ne reconnaît pas le protocole webcal.
Remplacer simplement webcal:// par https:// dans l'URL ci-dessous.
